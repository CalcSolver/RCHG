# Remote Control Hand Gestures 🌌

An ultra-low latency, computer-vision desktop peripheral control utility utilizing native OS streams.

## Features
* **Wand Tracking:** Move the pointer precisely using the midpoint vector of your index and middle fingers.
* **Stark Mode Toggle:** Switch seamlessly to Head/Eye tracking navigation interfaces.
* **Dynamic Audio Hooks:** Separate your fingers into a wide gesture structure to slide system volume layers up or down.

## Installation
1. Clone or download this repository.
2. Right-click and run `install.bat` as an Administrator to install package dependencies automatically.
3. Run `python gesture_controller.py` to start the app dashboard!
